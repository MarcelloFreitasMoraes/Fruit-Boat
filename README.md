# Barraca de Frutas

## Getting Started
The project runs on Node, React (with Next.js) and Styled Components.

## Installing
Clone the repository:
git clone https://github.com/caioSilva-hub/barraca-de-frutas.git

## Run npm install to install node dependencies:
npm i

## Start the project:
npm run dev
